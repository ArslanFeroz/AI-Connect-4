# Connect Four AI (Minimax)

This is a Python implementation of the classic **Connect Four** game where a human player competes against an AI.

The AI uses the **Minimax algorithm** with a heuristic evaluation function to make intelligent decisions.

---

## Features

- Fully playable Connect Four game in terminal
- Human vs AI gameplay
- AI powered by Minimax algorithm
- Board evaluation using heuristic scoring
- Detects:
  - Horizontal wins
  - Vertical wins
  - Diagonal wins (both directions)
- Input validation and error handling
